# strokeX ( One line puzzle )
a pretty fascinating game with an amazing neon GUI.

# How it works?
Open the executing file.
Here comes the front page with a unique neon mobile circles background.
Click start button.
Welcome to the gameplay.
Trace the given shape by connecting edges through nodes in one line without leaving the cursor and tracing twice.
Reset if you traced twice accidently.
Complete in minimum time to get the best score!
Levels uprgrade only after completing previous levels.

# Key features
Live effects when buttons hovered.
Shining glaze on tracing every edge.
Live timer for every level.
